# Perseus Technical Documentation

## Introduction
[Perseus](https://ib1.org/perseus) is a smart data scheme operated by Icebreaker One (IB1) that will help unlock access to finance that reduces emissions faster by automating sustainability reporting for every SME business in the UK.

This documentation describes the steps to become “Perseus Ready” as a data provider or data client within the Perseus scheme. This involves becoming legally ready by joining the Core Trust Framework and preparing to sign the Perseus Scheme Agreement, and technically ready by integrating with services in the Perseus Sandbox.

The Sandbox provides identical trust services (Registry and Directory) to Perseus in production. It includes a reference Energy Data Provider (EDP) that provides synthetic smart meter consumption data, and a reference Carbon Accounting Provider (CAP) to retrieve synthetic consumption and generate standardised emissions reports for Financial Service Providers (FSPs). It can safely be used for development and testing.
